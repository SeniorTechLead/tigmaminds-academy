# TigmaMinds Academy

STEM through imagination. 130+ stories from world traditions, each a doorway into real science, engineering, and code.

## What This Is

A story-driven learning platform where children explore STEM concepts through folklore and mythology — from Assamese tales to stories from Buddhist, Hindu, Islamic, and Christian traditions. Every story connects to real science: physics, biology, computer science, engineering.

## Stack

- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS
- **Auth & DB**: Supabase (PostgreSQL + Google OAuth)
- **Deployment**: Vercel
- **Icons**: Lucide React
- **Code Playground**: Pyodide (browser-based Python)

## Development

```bash
npm install
npm run dev
```

## Environment Variables

```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_anon_key
```
